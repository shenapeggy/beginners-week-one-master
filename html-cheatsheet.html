Coding For Girls: Cheat Sheet

Starting Layout

<!DOCTYPE html>

<html>

<head>

<title> name of window </title>

</head>


<body>


//body content


</body>

Body

Headings:

  • <h1> page title </h1>
  • ...
  • <h6> smallest (h4-6 probably wont see) </h6>

Paragraphs:

<p> .... </p>

Blank lines & void tags

<br> : break/blank lines (void tag: you don't need a closing tag)

Comments

<!-- comment -->

Lists

  • <ol> - ordered/numbers
  • <ul> - unordered/points
    • <li> - list items

Links

<a href="link"> word that is linked </a>

Images

<img> = "link/image source" alt="description" width="200px"> (void tags so no close).

Tables

<table class='table'>

<tbody>

<tr> - first row

<th> cell heading </th>

<td> cell </td>

</tr>

</tbody>

</table>
